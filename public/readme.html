Home Page

1.Fill out the information and click Insert to insert a student info into database.
2.Enter a name to search, then edit or delete.
3.Click Display to display all students (as rectangles with rounded corners for females).

Edit Page

1.Fill out the information and click update to modify a student info.
2.Click Delete to delete this searched student.
3.Again, Click Display to display all students.

Display Page

1.Select a student to edit in the Edit Page.
2.Students are represented as rectangles with rounded corners for females.
